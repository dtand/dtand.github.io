## Overview
A year-in-review analytics dashboard for GRC data.

## Technologies Used
- **JavaScript React w/ Typescript**: Frontend application
- **Golang**: backend API
- **Python**: utility scripts
- **Postgres**: database persistence layer

## Description
GRC Wrapped is a full stack application which aims to compile and normalize historical running club data and present it in a meaningful way.  The application defines and implements a large API spec for various functionality and also performs capabilities for parsing and injecting new performance data for athletes.  The backend integrates with the Anthropic API and uses Claude to help parse out key data from unstructured email summaries.

## Key Features
- Dashboard
- Athlete Performances
- REST API Layer
- Claude API
- Data Injection
